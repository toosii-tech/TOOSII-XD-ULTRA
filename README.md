# TOOSII-XD ULTRA

WhatsApp Multi-Device Bot built with gifted-baileys.

## Setup

1. Clone this repo
2. Run npm i
3. Run node index.js
4. Enter your WhatsApp number when prompted
5. Enter the pairing code on your phone

## Features

- AI Chat (Pollinations)
- Media Download (YouTube, Instagram, Facebook)
- Sticker Creation
- Group Management
- Games and Leaderboards
- Anti-link, Anti-tag, Anti-sticker
