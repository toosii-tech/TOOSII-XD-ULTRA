## Requirements

- Node.js v18 or higher
- RAM: Minimum 512MB (2GB recommended)
- Stable internet connection

## Installation

### Local Installation

```bash
# Clone the repository
git clone <repo-url>
cd toosii-xd-ultra

# Install dependencies
npm install

# Start the bot
npm start
```

### Deployment Options

This bot supports deployment on multiple platforms:

#### Heroku
```bash
heroku create your-app-name
git push heroku main
```

#### Railway
1. Connect your GitHub repository
2. Deploy automatically

#### Render
1. Create a new Web Service
2. Connect your repository
3. Set build command: `npm install`
4. Set start command: `npm start`

#### Koyeb
1. Create a new App
2. Select your repository
3. Configure and deploy

#### Panel Hosting (Pterodactyl, etc.)
1. Upload all files to your panel
2. Set startup command: `npm start`
3. Node.js 18+ required

## Bot Commands

Type `.menu` to see all available commands:

- `.menu ai` - AI features
- `.menu downloader` - Download media
- `.menu sticker` - Sticker tools
- `.menu group` - Group management
- `.menu games` - Fun games
- `.menu tools` - Utility tools
- `.menu search` - Search features
- `.menu owner` - Owner commands
